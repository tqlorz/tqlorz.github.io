大一上课程总结

写在前面

开个课程总结的坑，不知道后续能不能坚持填坑，这里仅仅写一些课程感想 （写给自己看的），比较随意。

数学分析（甲）Ⅰ（H） \(~~~\) 5.5 学分 贾厚玉

先前在看高等数学先修课的时候，视频种 jhy 老师总是青着个脸，不苟言笑的授课，让我看到预置课表的时候还有一些不安。不过实际上老师是很幽默风趣的，上课用的是 PPT + 电子板书，因此授课速度也比较快。上课总是拓展物理方面的数学知识😛，因为班上有求物和强物的同学。授课顺序是从集合 -> 数列极限 -> 函数极限，函数里又包括了连续、微分、积分的知识。

分析学的核心是非常朴素的极限的思想，也就是我们常说的 \(\epsilon - N\) 或者 \(\epsilon - \delta\) 语言。从简单的极限思想出发，能够生出许多概念与定义。我们用这些语言能清晰地描述出 “无穷” 这一定义，避免 “幽灵般的 \(\Delta x\) ” 这样的混淆，我想这就是数学分析最为吸引我的地方。

线代代数 Ⅰ（H） \(~~~\) 3.5 学分 刘康生

美好大学生活的第一节课，早八，就遇见了 lks 的线代。记忆犹新的是，因为疫情，老师的前几节课都是线上上课。但是老师似乎不会用 PPT 的公式编辑器，PPT 上全是公式的源码😅，再加上 Honor Hard 课的线代授课顺序有所不同，是从集合 -> 线性空间 -> 线性映射 -> 矩阵 -> 行列式 -> 特征理论，导致这门课真是给刚迈入大学的我迎头痛击。不过换个角度想，这门课让我知道了即使在 ZJU，自学能力也是大学学习的重要能力。

老师上课有一种独自沉浸在数学世界的美（大雾），再加上口齿不清和奇怪的口癖，导致学生几乎很难跟上他的思路。因此这门课成了我每周花费时间最长的课程，往往是课前过一遍书，课上听一遍，课后再看智云课堂回放。学习过程中，我也发现我对线代有很强的兴趣，我想这也是我学习线代的动力。课前看过一遍书，基本搞懂了以后，上课大部分时间也能跟上老师的思路了，其实发现他讲的还挺有道理🤯。不管如何，我对 H 课的线代还是肯定的，从抽象的事物出发联系到具象的代数，世上一切线性的东西都能与我笔尖上方整的矩阵同构，再找到它最简单的形式，剖析出种种本质来，我想这或许就是线性代数对我来说最令人难以拒绝的地方吧。

普通化学（H） \(~~~\) 3 学分 聂晶晶

普化真的是槽点特别多的一门课，“普通”两字就说明了这门课泛泛而谈的毛病，所涉及的内容极其广泛：化学热力学、电化学、化学动力学、结构化学，每一部分都大概叙述一遍，定性的概念远远多于定量的计算，对现象的描述也远远多于对本质的阐述，导致给人一种逛科技馆的感觉🙉。

对于老师的授课，嗯。。。有一些混乱，不过这其中也有一部分普化本身逻辑性就不强的原因吧。

工程图学（H） \(~~~\) 2.5 学分 费少梅

据说是画图天赋检验课，但其实这课给我的体验不错，fsm 老师十分耐心 （有着种回到小学的感觉），老师的引导也很循循渐进，体系也比较清晰明确。这门课本身与高中的通用技术还是有不同的，毕竟不只是三视图补线。立体的相贯线确实让人非常头疼，后面的 CAD 绘图作业也是十分的耗费时间，好在我对此不算十分讨厌，用尺规画出规整的图形还是感觉不错滴。

程序设计基础 \(~~~\) 3 学分 任重

首先，ZJU 的C语言基础课是一言难尽的，其中教考分离是最令人诟病的地方。程设的学习必然重在实验，但期末考试理论考占有很大比重，并且其中有很多不必要的、实际编程中根本不会发生的理论极端，导致这门课变得和大计基一样。

其次，虽然这门课在教学管理平台上标注的是双语课程，但是实际教学过程中往往是中文授课 + 英文PPT，但是我认为我们老师在上课过程中有许多不必要的中英混杂，我觉得这真的很具有迷惑性，并且我想在课上加入一些代码实操的部分或许会对学生理解更有帮助，而不是学了理论后在编程过程中自己摸黑。

中国近现代史纲要 \(~~~\) 3 学分 何惧

老师是刚来 ZJU 任教的，据他所说，我们是他的 “黄埔二期” 😜。毕竟是年轻老师，上课并不是照本宣章，而是有自己的清晰思路。老师布置的作业也和其他老师不太一样，是对家人做一次人生访谈，了解祖父辈过去的岁月，写下自己的感受和看法。总的来说，这门课的体验还是不错的，但并不能阻止学期后期我在课上看数分和线代🥺。

思想道德与法治 \(~~~\) 3 学分 陈小英

众所周知，思修是一门思政课，所以我 ______ 。

批判性思维 \(~~~\) 3 学分 王彦君

究极大氵课无疑，作为一门三分的通识课，没有任何大作业，平常作业也几乎为零，只有一场半小时就能做完的期末考试🥰。但相应的是，上课内容很单一笼统。虽然到现在我也没做到 “批判性思维” ，不过通识课混混分就行啦（×）。
